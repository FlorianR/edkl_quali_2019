## Callsign IDs
Hexadecimal, start with "06" for gliders, no noTrack, no stealth.

STttttaa

- S: Stealth
- T: NoTrack
- tttt: Aircraft type (see below)
- aa: address type (0 RANDOM, 1 ICAO, 2 FLARM, 3 OGN)

Aircraft types:
- 0: Unknown
- 1: glider
- 2: tow plane
- ...
